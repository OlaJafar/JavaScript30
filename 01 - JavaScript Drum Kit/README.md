# JavaScript Drum Kit

A simple JavaScript drum kit project that plays sounds when you press keyboard keys.

## How it works

- Each key on the keyboard (A, S, D, F, G, H, J, K, L) is connected to a drum sound.
- When you press a key, the sound plays.
- The key on the screen gets a small animation.
- After the animation ends, the style goes back to normal.

## Technologies used

- HTML
- CSS
- JavaScript
- DOM events
- Audio element

## Notes

This project is good for practicing:

- DOM manipulation
- Keyboard events
- Audio in JavaScript
- CSS transitions
